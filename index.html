Overview

Packgod Browser is a lightweight desktop browser built with Electron, focusing on fast startup, custom UI behavior, and modular code structure.

  • Custom boot screen system
  • Audio engine with event triggers
  • Minimal renderer architecture
  • Cross-platform builds (Windows + Linux)
